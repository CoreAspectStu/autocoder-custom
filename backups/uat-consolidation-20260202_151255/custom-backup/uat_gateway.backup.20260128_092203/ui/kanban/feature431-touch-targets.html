Feature #431: Touch Targets

WCAG 2.1 SC 2.5.5 - Minimum 44x44 pixels

WCAG 2.1 Success Criterion 2.5.5

Target Size (Level AAA)
  • Requirement: Touch targets must be at least 44x44 CSS pixels
  • Purpose: Ensure people with motor impairments can easily activate targets
  • Exceptions: Inline text, user agent controls, essential presentations
  • Spacing: Minimum 8px between adjacent targets
  • Benefits: Easier for everyone, especially on mobile devices

Why 44x44px? This size matches the average finger pad size and ensures users can accurately tap targets without accidental activations.
