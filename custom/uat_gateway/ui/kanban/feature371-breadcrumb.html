About Breadcrumb Navigation

Breadcrumbs provide a secondary navigation scheme that reveals the user's location in a website or web application. They should:

  • Show the hierarchy of the current page
  • Provide clickable links to parent pages
  • Update automatically as you navigate
  • Be accurate and reflect the true path
